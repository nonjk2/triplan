🎮 Triplan 주요 기능

    1. 지도 검색 (네이버 맵)
    2. 여행 계획 작성(날짜, 시간, 위치 등)
    3. 계획을 한 눈으로 볼수 있게(가시적으로)지도에서 핀으로 경로를 알 수 있음.
    4. 친구 초대를 통한 여행 계획 공유
